# 🐞 Defect Management Dashboard

A modern and responsive Defect Management System built using React, Node.js, Express, and MySQL. The application helps teams manage, track, analyze, and monitor software defects efficiently through powerful analytics, audit logs, notifications, and advanced filtering.

---

## ✨ Features

### 📋 Defect Management
- Create Defects
- Update Defects
- Delete Defects
- View Defect History Timeline
- Search & Filter Defects

### 📊 Analytics Dashboard
- Severity Analysis
- Aging Analysis
- Defect Source Distribution
- Workflow Pulse
- Environment Analysis
- Assigned To Analytics
- Added By Analytics
- Attention Required Metrics

### 🔍 Advanced Filtering

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/defect-management-dashboard.git
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm start
```

### Database

Create a MySQL database and configure your `.env` file.

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=defect_management
```
